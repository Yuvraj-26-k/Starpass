# 🔐 StarkPass v1.0

StarkPass is a beginner-friendly password generator built with Python. It allows users to generate secure random passwords of custom lengths using uppercase letters, lowercase letters, numbers, and special characters.

## Features
✅ Custom password length
✅ Random password generation
✅ Letters, numbers, and symbols
✅ Clean terminal interface
✅ Linux executable support

## Technologies Used
- Python 3
- random module

## What I Learned
- Python loops
- User input handling
- String manipulation
- Random data generation
- Basic project structure

## Usage
Run the script and enter the desired password length. StarkPass will generate a random secure password instantly.
